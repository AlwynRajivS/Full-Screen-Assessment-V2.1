Exam Details
Overview of the examination structure
  • The test consists of 50 MCQ questions — aptitude + department-specific.
  • Total duration is 90 minutes. Timer begins immediately after you enter the exam.
  • Questions are randomised and shuffled uniquely for each candidate.
  • There is no negative marking. Attempt all questions.
  • Each question carries 1 mark. Maximum score: 50.
🗺️
Navigation & Answering
How to navigate through questions
  • Save & Next — saves your answer and moves to the next question.
  • Mark for Review — flags the question (yellow) to revisit later.
  • Question Palette — colour-coded status: green = answered, yellow = review, purple = answered + review, grey = unanswered.
  • You can change your answer at any time before final submission.
  • Use Review All to see a summary of your answers before submitting.
⚠️
Violation Rules
Actions that trigger a violation warning
🔄 Refreshing or reloading the page
🪟 Switching tabs or windows
📋 Copy / Paste / Cut actions
🖥️ Exiting full-screen mode
📱 Long-press on screen (mobile)
🖱️ Right-click context menu
🔇 Minimising the exam window
📷 PrintScreen key press
🚨
Each violation is recorded in real time. Accumulating 20 violations will trigger an automatic submission of your test immediately. Returning to a refreshed/closed page also triggers auto-submission.
✅
Submission Rules
Final submission guidelines
  • The system will auto-submit when the 90-minute timer expires.
  • You may manually submit at any time using the "Submit" button on the last question.
  • After submission, re-login is not possible. Your session is permanently locked.
  • Results are saved securely on the server immediately upon submission.
  • If the network fails at submission, results are saved locally and retried automatically.
